## Quick Summary

**Stack:** Solid.js + TypeScript + Vite + Tailwind CSS + pretext + SurrealDB Wasm + Transformers.js + Tone.js

**Unique features:**
- Dead Text Graveyard (deleted text as a creative journal)
- Dual Pulse (writer's ECG + story's ECG)
- Text Sonification (the rhythm of text as sound)
- Ghost Reader (simulated first reader)
- Character Simulator ("What if...")

**Current phase:** Phase 1 — Proof of Concept

**Core principle:** Everything runs in the browser. Offline-first. Zero server dependency.
